# Overview

This repository contains multiple graphical user interface (GUI) projects developed in Python. Every project demonstrates different concepts of desktop application development, ranging from simple interfaces to more advanced applications.

The goal of this repository is to provide clean, understandable, and reusable examples for anyone interested in Python GUI development.

# Installation

Clone the repository

```bash
git clone https://github.com/USERNAME/REPOSITORY.git
```

Enter the project folder

```bash
cd REPOSITORY
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
